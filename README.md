# Artist Management System API

## Tech Stack

Node.js, Express, TypeScript, PostgreSQL (Knex), JWT, Resend, Zod, Multer, csv-parse, csv-stringify

## Install & Run

```bash
npm install
cp .env.example .env   # Update the env variables
npm run migrate:latest # Run database migrations
npm run dev            # Start development server with hot-reload
npm run build          # Compile to dist/
npm start              # Start production server
```
